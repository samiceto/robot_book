# Chapter 19: Human-Robot Interaction

## Quick Start

```bash
# Install dependencies
pip install speechrecognition pyttsx3 mediapipe

# Run multimodal interface
python3 multimodal_interface.py

# Voice-only control
python3 voice_control.py
```

## Modalities

- Speech recognition (Google/Whisper)
- Text-to-speech (pyttsx3)
- Gesture recognition (MediaPipe)
- Safety monitoring

## Performance

- Speech recognition: 70-80% accuracy
- Gesture detection: 30 FPS
- Response time: <500ms
