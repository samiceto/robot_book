# Chapter 20: Safety and Compliance

## Quick Start

```bash
# Run safety system tests
python3 test_safety.py

# Collision detection demo
ros2 run chapter_20 collision_monitor

# Emergency stop test
python3 test_estop.py
```

- Collision detection (torque-based)
- Emergency stop (hardware + software)

- HAZOP risk assessment templates

## Compliance

- ISO 13482 checklist
- Risk assessment documentation
- Test reports
